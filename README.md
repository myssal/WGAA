# WGAA Museum

Asset viewer website for [Punishing Gray Raven](https://pgr.kurogame.net).

## Features:
 - Multi-servers data support: `EN` and `CN`.
 - Contents:
    + [CG](#cg)
    + [Manga](#manga)
    + [Emoji](#emoji)
    + [Story Sprite](#story-sprite)
    + [Memory](#memory)
    + [Coating](#coating)
